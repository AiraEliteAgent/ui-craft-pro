# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-03-21

### Added
- Initial OpenClaw-compatible skill packaging for `ui-craft-pro`
- Core skill instructions in `SKILL.md`
- Local design knowledge base in `data/`
- Search and design-system scripts in `scripts/`
- Implementation guidance in `references/implementation-patterns.md`
- Review guidance in `references/review-checklist.md`
- Product-archetype correction guidance in `references/product-archetypes.md`

### Changed
- Renamed the working project to `ui-craft-pro`
- Reframed the skill from design-reference lookup into a **design-to-implementation** workflow
- Strengthened instructions around:
  - locking implementation choices before coding
  - correcting emotionally wrong first-pass design output
  - reviewing output for consistency and style fit

### Notes
- This version was iterated through live demos and cross-agent testing.
- The project includes inspiration and adapted ideas from the original `ui-ux-pro-max-skill` project, but is shaped for OpenClaw-based use.
